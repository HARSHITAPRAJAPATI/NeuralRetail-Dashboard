## 📌 Project Overview

NeuralRetail is an end-to-end data science project focused on retail analytics.
It provides actionable business insights through interactive dashboards and machine learning techniques.

---

## 🎯 Key Features

### 📊 Sales Analytics

* Daily sales trend visualization
* Moving average analysis
* Country-wise revenue insights

### 🛍️ Product Insights

* Top-selling products
* Revenue contribution analysis

### 👥 Customer Segmentation (RFM)

* Recency, Frequency, Monetary analysis
* Customer grouping (VIP, Loyal, At Risk, Lost)

### ⚠️ Churn Analysis

* Identifies potential churn customers
* Helps improve retention strategies

### 📦 Inventory Optimization

* Safety Stock calculation
* Reorder Point analysis
* Risk identification

### 🎛️ Interactive Dashboard

* Built using Streamlit
* Dynamic filters (Country, Date)
* Real-time insights

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/HARSHITAPRAJAPATI/NeuralRetail-Dashboard.git
cd NeuralRetail-Dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```
